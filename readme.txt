== Description ==

Features include:

* SEO assistance
* Customizer import export options
* Options toggle for Embed and emoji scripts

== Frequently Asked Questions ==

= Is this a required plugin for Greenlet theme =

No. Greenlet theme works without this plugin. But we recommend to install this plugin to add more options to the theme as well as improve performance.
